Why is the live sandbox not up yet?
MetaSift runs a full OpenMetadata stack (server + MySQL + Elasticsearch) plus the FastAPI backend, the React SPA, and a daily reset job. We're hardening the deploy — rate-limiting, BYO-key gating, read-only mode — before exposing it publicly. Subscribe to the repo for the launch.

Run it locally

The fastest path. Boots the OpenMetadata stack, seeds a messy demo catalog, and starts the API + UI. Requires Docker and Python 3.11. 

# Clone the repo
$ git clone https://github.com/blueberrylinux/metasift.git && cd metasift

# Install deps + boot OpenMetadata + seed the demo catalog
$ make install && make stack-up && make seed

# Start FastAPI on :8000 and the React dev server on :5173
$ make api
$ cd web && npm run dev

# → open http://localhost:5173

What you'll see

A composite-score dashboard, eleven visualization tabs, and an LLM-powered agent — all grounded in the real OpenMetadata REST API, not mocks.

Composite Score analysis
One number that captures coverage (30%) + accuracy (30%) + classification consistency (20%) + description quality (20%). Documentation coverage alone lies; this doesn't.
Cleaning engine differentiator
Detects stale descriptions, scores quality 1–5, flags tag conflicts across schemas, and clusters fuzzy-matched names like cust_id / cid / customer_id.
Stew the agent 30 tools
LangChain agent over 27 local tools + 3 MCP tools. Every reply ships with a "Show your work" expander so you can audit which tools fired and what they returned.
DQ track 3 hackathon issues
Plain-English explanations of failing checks, AI-generated test recommendations, and a risk score that joins failing tests × lineage × PII downstream.
Review queue no silent writes
Every suggestion — descriptions, PII tags, ownership — flows through accept / edit / reject before the catalog gets touched. Auditable by design.
11 viz tabs interactive
Score gauge, lineage DAG, governance overlay, blast radius, stewardship by team, catalog map, tag conflicts, quality, and the full DQ trio.
Built for the WeMakeDevs × OpenMetadata "Back to the Metadata" hackathon.
